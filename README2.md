Amazon Sales Analysis Dashboard (Power BI)

Project Overview

This project presents an interactive Power BI dashboard built using an Amazon Sales dataset.
The dashboard focuses on sales performance, order trends, product analysis, and regional insights, without using a profit column.
It is designed to help stakeholders understand what is selling, where it is selling, and how orders are being fulfilled.

Dashboard Objectives

Analyze overall sales and order performance
Identify top-performing states, categories, and SKUs
Track monthly sales and order trends
Understand fulfilment and operational metrics
Provide an executive-level sales overview

Tools & Technologies Used

Power BI Desktop
DAX (Data Analysis Expressions)
Power Query for data cleaning
CSV dataset (Amazon Sales Report)
